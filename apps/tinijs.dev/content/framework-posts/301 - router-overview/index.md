**Tini Router** is a way to add routing capability to TiniJS apps. It has several useful features:

- **Bundle** or **lazy load** pages
- Routes with **layouts**
- Many **param patterns**
- Navigate using the `a` tag
- Route **guards**
- **404** pages
- _And more_


To get started, install:

```bash
npm i @tinijs/router
```
