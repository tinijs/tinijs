Watch for changes and rebuild without development server.

```bash
npx tini watch
```

## Options

| Option | Description |
| --- | --- |
| `--lazy` or `-l` | No initial build, only build for new changes. |
| `--clean` or `-c` | Clean the output dir before build. |
