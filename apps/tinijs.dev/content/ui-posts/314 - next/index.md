NextUI-inspired theme family, homepage: https://nextui.org/

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-next/bundled.js
```

- NPM:

```bash
npm i @tinijs/ui-next
```

- React NPM:

```bash
npm i @tinijs/ui-next-react
```
