Ant-inspired theme family, homepage: https://ant.design/

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-ant/bundled.js
```

- NPM:

```bash
npm i @tinijs/ui-ant
```

- React NPM:

```bash
npm i @tinijs/ui-ant-react
```
