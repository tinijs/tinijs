Daisy-inspired theme family, homepage: <https://daisyui.com>

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-daisy/bundled/setup.js
```

- NPM:

```bash
npm i @tinijs/ui-daisy
```

- React NPM:

```bash
npm i @tinijs/ui-daisy-react
```
