Radix-inspired theme family, homepage: https://www.radix-ui.com/

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-radix/bundled.js
```

- NPM:

```bash
npm i @tinijs/ui-radix
```

- React NPM:

```bash
npm i @tinijs/ui-radix-react
```
