Prime-inspired theme family, homepage: https://primeng.org/

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-prime/bundled.js
```

- NPM:

```bash
npm i @tinijs/ui-prime
```

- React NPM:

```bash
npm i @tinijs/ui-prime-react
```
