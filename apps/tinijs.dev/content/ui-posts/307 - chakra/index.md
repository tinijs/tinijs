Chakra-inspired theme family, homepage: <https://v2.chakra-ui.com>

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-chakra/bundled/setup.js
```

- NPM:

```bash
npm i @tinijs/ui-chakra
```

- React NPM:

```bash
npm i @tinijs/ui-chakra-react
```
