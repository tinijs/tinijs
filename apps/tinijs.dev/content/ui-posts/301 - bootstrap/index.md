Bootstrap-inspired theme family, homepage: https://getbootstrap.com/

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-bootstrap/bundled.js
```

- NPM:

```bash
npm i @tinijs/ui-bootstrap
```

- React NPM:

```bash
npm i @tinijs/ui-bootstrap-react
```
