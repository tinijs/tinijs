## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-horizon/bundled/setup.js
```

- NPM:

```bash
npm i @tinijs/ui-horizon
```

- React NPM:

```bash
npm i @tinijs/ui-horizon-react
```
