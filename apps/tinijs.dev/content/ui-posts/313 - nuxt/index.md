NuxtUI-inspired theme family, homepage: https://ui.nuxt.com/

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-nuxt/bundled/setup.js
```

- NPM:

```bash
npm i @tinijs/ui-nuxt
```

- React NPM:

```bash
npm i @tinijs/ui-nuxt-react
```
