iOS-inspired theme family, homepage: <https://developer.apple.com/design/human-interface-guidelines>

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-ios/bundled/setup.js
```

- NPM:

```bash
npm i @tinijs/ui-ios
```

- React NPM:

```bash
npm i @tinijs/ui-ios-react
```
