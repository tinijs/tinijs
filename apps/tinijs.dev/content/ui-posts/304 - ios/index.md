iOS-inspired theme family, homepage: <https://developer.apple.com/design/human-interface-guidelines>

## Install

- CDN:

```txt
https://cdn.jsdelivr.net/npm/@tinijs/ui-ios@0.21.0/bundled/setup.js
```

- NPM:

```bash
npm i @tinijs/ui-ios
```

- React NPM:

```bash
npm i @tinijs/ui-ios-react
```

- CLI:

```js
{
  sources: ['@tinijs/ui'],
  families: {
    ios: ['light', 'dark'],
  }
}
```
