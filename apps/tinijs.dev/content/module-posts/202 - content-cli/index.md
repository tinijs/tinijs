## `content build`

The build content of the [Tini Content](/module/content) module.

```bash
npx tini content build
```

| Option | Description |
| --- | --- |
| `--dir <value>` or `-d` | Custom source dir, default to `content`. |
| `--outDir <value>` or `-o` | Custom out dir, default to `app/public`. |
